Métodos para Calcular Límites

Teoría: Método de Sustitución Directa

Se sustituye directamente el valor al que x tiende en la función, siempre que ésta sea continua. Es el método más simple para límites de funciones polinómicas, exponenciales o trigonométricas continuas.

Ejemplos

Ejemplo 1: limx → 3 (2x + 5)

  1. La función es un polinomio y continua → podemos sustituir directamente.
  2. 2·3 + 5 = 11 → límite = 11.

Ejemplo 2: limx → 0 sin(x)

  1. sin(x) es continua en 0 → sustituimos directamente: sin(0) = 0
  2. Por lo tanto, límite = 0.

Ejemplo 3: limx → 1 e^x

  1. La función exponencial e^x es continua → sustituimos x = 1
  2. e^1 = e → límite = e

Teoría: Factorización / Método de Ruffini

Se usa cuando sustituir directamente genera una indeterminación 0/0. Se factoriza el polinomio o se aplica Ruffini para simplificar y eliminar el factor que produce la indeterminación.

Ejemplos

Ejemplo 1: limx → 2 (x² - 4)/(x - 2)

  1. Sustituir x = 2 → 0/0 → indeterminación.
  2. Factorizamos: x² - 4 = (x - 2)(x + 2)
  3. Cancelamos x - 2 → queda x + 2
  4. Sustituimos x = 2 → 4 → límite = 4

Ejemplo 2: limx → 1 (x³ - 1)/(x - 1)

  1. Sustituir x = 1 → 0/0
  2. Factorizamos usando Ruffini: x³ - 1 = (x - 1)(x² + x + 1)
  3. Cancelamos x - 1 → queda x² + x + 1
  4. Sustituimos x = 1 → 1 + 1 + 1 = 3 → límite = 3

Ejemplo 3: limx → -1 (x² + 3x + 2)/(x + 1)

  1. Sustituir x = -1 → 0/0
  2. Factorizamos: x² + 3x + 2 = (x + 1)(x + 2)
  3. Cancelamos x + 1 → queda x + 2
  4. Sustituimos x = -1 → -1 + 2 = 1 → límite = 1

Teoría: Racionalización

Se aplica cuando hay raíces cuadradas que generan indeterminación 0/0. Multiplicamos por el conjugado para eliminar la raíz y simplificar.

Ejemplos

Ejemplo 1: limx → 4 (√x - 2)/(x - 4)

  1. Sustituir x = 4 → 0/0
  2. Multiplicamos por el conjugado: (√x + 2)
  3. Simplificamos: [(√x - 2)(√x + 2)] / [(x - 4)(√x + 2)] = (x - 4)/(x - 4)(√x + 2) = 1/(√x + 2)
  4. Sustituimos x = 4 → 1/4 → límite = 1/4

Ejemplo 2: limx → 9 (√x - 3)/(x - 9)

  1. Sustituir x = 9 → 0/0
  2. Multiplicamos por el conjugado: (√x + 3)
  3. Simplificamos: (x - 9)/(x - 9)(√x + 3) = 1/(√x + 3)
  4. Sustituimos x = 9 → 1/6 → límite = 1/6

Ejemplo 3: limx → 1 (√(x + 3) - 2)/(x - 1)

  1. Sustituir x = 1 → 0/0
  2. Multiplicamos por el conjugado: (√(x + 3) + 2)
  3. Simplificamos: [(x + 3) - 4]/[(x - 1)(√(x + 3) + 2)] = (x - 1)/[(x - 1)(√(x + 3) + 2)] = 1/(√(x + 3) + 2)
  4. Sustituimos x = 1 → 1/(2 + 2) = 1/4 → límite = 1/4

Teoría: Límites Laterales

Se estudian los límites desde la izquierda (x → a⁻) y desde la derecha (x → a⁺). Si coinciden, existe el límite; si no, el límite no existe.

Ejemplos

Ejemplo 1: limx → 0 |x|/x

  1. x → 0⁻ → |x|/x = -1
  2. x → 0⁺ → |x|/x = 1
  3. Los límites laterales no coinciden → límite no existe.

Ejemplo 2: limx → 2 (x² - 4)/(x - 2)

  1. Factorizamos: (x - 2)(x + 2)/(x - 2) → x + 2
  2. Como x → 2⁻ y x → 2⁺ ambos dan 4 → límite = 4

Ejemplo 3: limx → 1 (x - 1)/|x - 1|

  1. x → 1⁻ → (x - 1)/|x - 1| = -1
  2. x → 1⁺ → (x - 1)/|x - 1| = 1
  3. Los límites laterales no coinciden → límite no existe.

Teoría: Límites al Infinito

Se estudia cómo se comporta la función cuando x crece mucho (x → ∞) o decrece mucho (x → -∞). Útil para determinar asíntotas horizontales o crecimiento de la función.

Ejemplos

Ejemplo 1: limx → ∞ (2x² + 3)/(x² - 1)

  1. Dividimos por x² (mayor potencia): (2 + 3/x²)/(1 - 1/x²)
  2. Cuando x → ∞, 3/x² → 0 y 1/x² → 0
  3. Queda 2/1 = 2 → límite = 2

Ejemplo 2: limx → -∞ (3x³ - x)/(x³ + 2x)

  1. Dividimos por x³: (3 - 1/x²)/(1 + 2/x²)
  2. Cuando x → -∞ → 3/1 = 3 → límite = 3

Ejemplo 3: limx → ∞ 5x/(2x + 1)

  1. Dividimos numerador y denominador por x: 5/(2 + 1/x)
  2. Cuando x → ∞, 1/x → 0 → 5/2 → límite = 5/2
