Definición Formal de Límites

Teoría

La definición formal de límite permite demostrar rigurosamente que una función f(x) se aproxima a un valor L cuando x se aproxima a a.

Se enuncia así:

limx → a f(x) = L

Significa que:

Para todo número positivo ε (epsilon), por muy pequeño que sea, existe un número positivo δ (delta) tal que, siempre que 0 < |x - a| < δ, se cumple |f(x) - L| < ε.

En otras palabras:

  • ε representa la precisión deseada en el valor de la función.
  • δ representa cuán cerca debe estar x del punto a para garantizar esa precisión.
  • Si se puede encontrar un δ para cada ε, entonces el límite existe y es igual a L.

Esta definición es la base de la rigor matemático en cálculo y permite demostrar límites de forma exacta, no solo aproximativa.

Ejemplos

Ejemplo 1: limx → 2 (3x + 1) = 7

Explicación paso a paso:

  1. Queremos demostrar que para todo ε > 0 existe δ > 0 tal que |(3x + 1) - 7| < ε siempre que 0 < |x - 2| < δ.
  2. Calculamos |f(x) - L|: |(3x + 1) - 7| = |3x - 6| = 3|x - 2|
  3. Queremos que 3|x - 2| < ε → |x - 2| < ε/3
  4. Por lo tanto, podemos elegir δ = ε/3
  5. Con esta elección, se cumple la definición formal: para cualquier ε > 0, si |x - 2| < δ = ε/3, entonces |f(x) - 7| < ε

Ejemplo 2: limx → 1 x² = 1

Explicación paso a paso:

  1. Partimos de |f(x) - L| = |x² - 1| = |x - 1||x + 1|
  2. Queremos |x - 1||x + 1| < ε. Necesitamos controlar |x + 1|.
  3. Podemos restringir δ ≤ 1 → entonces |x - 1| < 1 → 0 < x < 2 → |x + 1| < 3
  4. Por lo tanto, |x - 1||x + 1| < 3|x - 1| < ε → necesitamos |x - 1| < ε/3
  5. Elegimos δ = min(1, ε/3). Con esto, se cumple la definición formal de límite.

Ejemplo 3: limx → 0 5x = 0

Explicación paso a paso:

  1. |f(x) - L| = |5x - 0| = 5|x|
  2. Queremos 5|x| < ε → |x| < ε/5
  3. Entonces basta elegir δ = ε/5
  4. Con esto, para cualquier ε > 0, si |x| < δ, se cumple |5x - 0| < ε → límite comprobado.
