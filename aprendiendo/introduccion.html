Introducción a los Límites

Teoría

En cálculo, un límite describe el valor al que se aproxima una función cuando la variable independiente se acerca a un cierto punto. Es fundamental para entender el comportamiento de las funciones cerca de puntos críticos, y es la base para derivadas e integrales.

Formalmente, decimos que:

Si f(x) se acerca a L cuando x se aproxima a a, entonces el límite de f(x) cuando x tiende a a es L.

Se escribe matemáticamente como:

limx → a f(x) = L

Algunos conceptos clave:

  • Límites laterales: El límite desde la izquierda (x → a⁻) y desde la derecha (x → a⁺) pueden diferir.
  • Indeterminaciones: A veces al sustituir directamente obtenemos formas como 0/0 o ∞/∞, que requieren técnicas especiales.
  • Límites infinitos: Cuando la función crece sin límite al acercarse a un punto o cuando x tiende a infinito.

Ejemplos

Ejemplo 1: Límite de una función polinómica

limx → 2 (3x² + 2x - 1)

Explicación paso a paso:

  1. Se trata de un polinomio. Los polinomios son continuos en todos los números reales, así que podemos sustituir directamente el valor de x.
  2. Sustituimos x = 2: 3·(2)² + 2·(2) - 1
  3. Calculamos paso a paso: 3·4 + 4 - 1 = 12 + 4 - 1 = 15
  4. Por lo tanto, el límite es 15.

Ejemplo 2: Límite que requiere simplificación

limx → 1 (x² - 1)/(x - 1)

Explicación paso a paso:

  1. Si sustituimos x = 1 directamente, obtenemos 0/0, lo que es una indeterminación. Esto significa que debemos simplificar la expresión antes de calcular el límite.
  2. Factorizamos el numerador: x² - 1 = (x - 1)(x + 1)
  3. Ahora la expresión queda: (x - 1)(x + 1) / (x - 1)
  4. Cancelamos el factor común (x - 1): la expresión se simplifica a x + 1
  5. Sustituimos x = 1: 1 + 1 = 2
  6. Por lo tanto, el límite es 2.

Ejemplo 3: Límite lateral

limx → 0⁺ 1/x

Explicación paso a paso:

  1. Este límite es lateral, indicado por 0⁺, lo que significa que x se aproxima a 0 desde valores positivos.
  2. Cuando x es un número muy pequeño y positivo (0.1, 0.01, 0.001…), 1/x se hace cada vez más grande.
  3. Por ejemplo: 1/0.1 = 10, 1/0.01 = 100, 1/0.001 = 1000…
  4. Vemos que la función crece sin límite, por lo que decimos que el límite es +∞.

Ejemplo 4: Límite con sustitución y fracción

limx → 3 (2x² - 5x + 1)/(x - 3)

Explicación paso a paso:

  1. Al sustituir x = 3, obtenemos 2·9 - 15 + 1 / (3-3) = (18 - 15 + 1)/0 = 4/0, que indica que el límite tiende a infinito o no existe. Debemos analizar el signo desde la izquierda y la derecha.
  2. Desde la derecha (x → 3⁺), el numerador ≈ 4, el denominador es pequeño positivo → límite +∞
  3. Desde la izquierda (x → 3⁻), el denominador es pequeño negativo → límite -∞
  4. Como los límites laterales no coinciden, el límite en x = 3 no existe.
